# machine-learning-portfolio
Hola, soy Johan Sebastián Lavacude Galvis, Ingeniero Industrial con interés en análisis de datos, machine learning y optimización de procesos.

Este repositorio recopila los proyectos desarrollados durante mi proceso de aprendizaje en ciencia de datos y machine learning, aplicando técnicas de análisis exploratorio, visualización de datos, modelado predictivo y evaluación de modelos.

## Tecnologías utilizadas
Python,
Pandas,
NumPy,
Matplotlib,
Seaborn,
Scikit-Learn,
Jupyter Notebook,

## Objetivo Profesional

Mi objetivo es desarrollar competencias en análisis de datos y machine learning para aplicarlas en entornos industriales y empresariales, generando valor mediante la optimización de procesos y la toma de decisiones basada en datos.
